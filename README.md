# Django 提供了一套功能完整的、灵活的、易于拓展的用户认证系统：django.contrib.auth 
## 主要功能
- 提供用户注册、登录、修改密码、找回密码等功能 
- 拓展 auth 内置的 User 模型  
## 所需环境
- Python == 3.6.1 
- Django == 2.1.3 
- pytz==2018.7
